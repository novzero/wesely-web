## ❓about the project
![wesely의 사본](https://github.com/guruma99/wesely-web/assets/121204952/730db0cc-3b06-428f-9a0e-e970e01a6b9a)

   - 현재위치(동네)기반의 운동시설 추천, 리뷰기능으로 사용자의 운동종목과 시설의 선택을 돕는다.
   - 커뮤니티를 통한 소통창구를 제공 및 비즈니스 계정을 통한 사업자의 운동시설 홍보를 돕는다.
